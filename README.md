# GBMU

Rust GameBoy Advanced Emulator

## Dependencies

### For Linux

Building this project on linux requires `cmake` and `gtk3`.

#### using apt

```sh
apt update && apt install cmake gtk3
```

#### using pacman

```sh
pacman -Sy cmake gtk3
```

### For MacOS

Building this project on MacOs requires `cmake`.

```sh
brew install cmake
```
